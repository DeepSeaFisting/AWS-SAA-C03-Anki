# AWS-SAA-C03-Anki
In-Scope AWS Services: 
📊 Analytics 

Amazon Athena: A serverless query service that lets you analyze data in S3 using SQL.  

AWS Data Exchange: A service that lets you find, subscribe to, and use third-party datasets in AWS.  

Amazon Data Firehose: A fully managed service for streaming data into AWS storage and analytics services.  

Amazon EMR: A managed big data platform for processing large datasets using frameworks like Hadoop and Spark.  

AWS Glue: A serverless data integration service for discovering, preparing, and combining data for analytics.  

Amazon Kinesis: A platform for collecting, processing, and analyzing real-time streaming data.  

AWS Lake Formation: A service for building, securing, and managing data lakes on AWS.  

Amazon MSK (Managed Streaming for Apache Kafka): A managed service for running Apache Kafka to process streaming data.  

Amazon OpenSearch Service: A managed service for search, log analytics, and real-time application monitoring.  

Amazon QuickSight: A business intelligence service for creating interactive dashboards and visualizations.  

Amazon Redshift: A fully managed data warehouse service for large-scale data analytics.  

🔗 Application Integration 

Amazon AppFlow: A service for securely transferring data between SaaS apps and AWS services.  

AWS AppSync: A managed GraphQL service for building real-time and offline-capable APIs.  

Amazon EventBridge: A serverless event bus for connecting applications using events.  

Amazon MQ: A managed message broker service supporting ActiveMQ and RabbitMQ.  

Amazon SNS: A pub/sub messaging service for sending notifications to multiple subscribers.  

Amazon SQS: A message queuing service for decoupling distributed applications.  

AWS Step Functions: A service for orchestrating workflows using state machines.  

💰 AWS Cost Management 

AWS Budgets: A tool to set custom cost and usage budgets and receive alerts.  

AWS Cost and Usage Report (CUR): A detailed report of AWS usage and costs for analysis.  

AWS Cost Explorer: A tool for visualizing and analyzing AWS spending over time.  

Savings Plans: A pricing model offering discounted rates in exchange for committed usage.  

🖥️ Compute 

AWS Batch: A service for running batch computing jobs at any scale.  

Amazon EC2: A service that provides resizable virtual servers in the cloud.  

Amazon EC2 Auto Scaling: A service that automatically adjusts EC2 capacity based on demand.  

AWS Elastic Beanstalk: A platform-as-a-service for deploying and managing applications.  

AWS Outposts: A service that brings AWS infrastructure and services on-premises.  

AWS Serverless Application Repository: A repository for discovering and deploying serverless applications.  

VMware Cloud on AWS: A service for running VMware workloads natively on AWS.  

AWS Wavelength: A service that delivers ultra-low latency applications using 5G networks.  

📦 Containers 

Amazon ECR: A managed container image registry.  

Amazon ECS: A managed container orchestration service.  

Amazon ECS Anywhere: A service to run ECS workloads on on-premises infrastructure.  

Amazon EKS: A managed Kubernetes service.  

Amazon EKS Anywhere: A way to run Kubernetes on-premises using EKS tooling.  

Amazon EKS Distro: A Kubernetes distribution used by EKS.  

🗄️ Database 

Amazon Aurora: A high-performance, MySQL- and PostgreSQL-compatible relational database.  

Amazon Aurora Serverless: An on-demand, auto-scaling version of Aurora.  

Amazon DocumentDB: A MongoDB-compatible managed NoSQL database.  

Amazon DynamoDB: A fully managed NoSQL key-value and document database.  

Amazon ElastiCache: A managed in-memory caching service using Redis or Memcached.  

Amazon Keyspaces: A serverless Cassandra-compatible database service.  

Amazon Neptune: A managed graph database service.  

Amazon RDS: A managed relational database service.  

Amazon Redshift: A data warehouse for analytics workloads.  

🛠️ Developer Tools 

AWS X-Ray: A service for analyzing and debugging distributed applications.  

🌐 Front-End Web and Mobile 

AWS Amplify: A platform for building and deploying full-stack web and mobile apps.  

Amazon API Gateway: A service for creating, publishing, and managing APIs.  

AWS Device Farm: A service for testing mobile apps on real devices.  

🤖 Machine Learning 

Amazon Comprehend: A service for natural language processing and text analysis.  

Amazon Kendra: An intelligent search service powered by machine learning.  

Amazon Lex: A service for building conversational chatbots using voice and text.  

Amazon Polly: A text-to-speech service.  

Amazon Rekognition: A service for image and video analysis.  

Amazon SageMaker: A platform for building, training, and deploying machine learning models.  

Amazon Textract: A service that extracts text and data from documents.  

Amazon Transcribe: A speech-to-text service.  

Amazon Translate: A neural machine translation service.  

⚙️ Management and Governance 

AWS Auto Scaling: A service that automatically adjusts resources across multiple services.  

AWS CLI: A command-line tool for managing AWS services.  

AWS CloudFormation: A service for provisioning infrastructure as code.  

AWS CloudTrail: A service that records AWS API activity for auditing.  

Amazon CloudWatch: A monitoring and observability service.  

AWS Compute Optimizer: A service that recommends optimal AWS resource configurations.  

AWS Config: A service for tracking resource configurations and compliance.  

AWS Control Tower: A service for setting up and governing multi-account AWS environments.  

AWS Health Dashboard: A service providing alerts and status on AWS service issues.  

AWS License Manager: A service for managing software licenses.  

Amazon Managed Grafana: A managed service for visualizing metrics and logs.  

Amazon Managed Service for Prometheus: A managed monitoring service for Prometheus metrics.  

AWS Management Console: A web interface for managing AWS resources.  

AWS Organizations: A service for managing multiple AWS accounts centrally.  

AWS Service Catalog: A service for creating and managing approved resource catalogs.  

AWS Systems Manager: A service for managing and automating infrastructure operations.  

AWS Trusted Advisor: A service that provides best practice recommendations.  

AWS Well-Architected Tool: A tool for reviewing architectures against AWS best practices.  

🎬 Media Services 

Amazon Elastic Transcoder: A service for converting media files into different formats.  

Amazon Kinesis Video Streams: A service for securely streaming video data.  

🚚 Migration and Transfer 

AWS Application Migration Service: A service for migrating applications to AWS with minimal downtime.  

AWS DataSync: A service for transferring large amounts of data to and from AWS.  

AWS DMS: A service for migrating databases to AWS.  

AWS Snow Family: A set of physical devices for data transfer and edge computing.  

AWS Transfer Family: A service for transferring files using SFTP, FTPS, and FTP.  

🌍 Networking and Content Delivery 

AWS Client VPN: A managed VPN service for secure remote access.  

Amazon CloudFront: A global content delivery network (CDN).  

AWS Direct Connect: A dedicated network connection from on-premises to AWS.  

Elastic Load Balancing (ELB): A service that distributes traffic across multiple targets.  

AWS Global Accelerator: A service that improves application availability and performance using AWS global network.  

AWS PrivateLink: A service for private connectivity between VPCs and AWS services.  

Amazon Route 53: A scalable DNS and domain name service.  

AWS Site-to-Site VPN: A VPN connection between on-premises and AWS.  

AWS Transit Gateway: A service that connects multiple VPCs and networks.  

Amazon VPC: A logically isolated virtual network in AWS.  

🔐 Security, Identity, and Compliance 

AWS Artifact: A service for accessing AWS compliance reports.  

AWS Audit Manager: A service that automates audit evidence collection.  

AWS Certificate Manager (ACM): A service for provisioning and managing SSL/TLS certificates.  

AWS CloudHSM: A hardware security module service for managing encryption keys.  

Amazon Cognito: A service for user authentication and access control.  

Amazon Detective: A service for analyzing and investigating security findings.  

AWS Directory Service: A managed directory service for Microsoft AD and other directories.  

AWS Firewall Manager: A service for centrally managing firewall rules.  

Amazon GuardDuty: A threat detection service.  

AWS IAM Identity Center: A service for centralized access management (SSO).  

Amazon Inspector: A service for automated security assessments.  

AWS KMS: A service for managing encryption keys.  

Amazon Macie: A service for discovering and protecting sensitive data.  

AWS Network Firewall: A managed network firewall service.  

AWS RAM (Resource Access Manager): A service for sharing AWS resources across accounts.  

AWS Secrets Manager: A service for securely storing and retrieving secrets.  

AWS Security Hub: A service for centralized security findings.  

AWS Shield: A managed DDoS protection service.  

AWS WAF: A web application firewall service.  

IAM: A service for managing access to AWS resources.  

⚡ Serverless 

AWS AppSync: A GraphQL API service with real-time capabilities.  

AWS Fargate: A serverless compute engine for containers.  

AWS Lambda: A serverless compute service that runs code in response to events.  

💾 Storage 

AWS Backup: A service for centrally managing backups.  

Amazon EBS: A block storage service for EC2.  

Amazon EFS: A scalable file storage service.  

Amazon FSx: A managed file system service supporting multiple file system types.  

Amazon S3: An object storage service.  

Amazon S3 Glacier: A low-cost storage service for archival data.  

AWS Storage Gateway: A hybrid storage service connecting on-premises systems to AWS.
